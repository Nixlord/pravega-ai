# pravega-ai
AI models on Heroku

## Scripts
Start Prod Server: `npm run serve-prod`
<br />

Start Dev Servers:
* React Dev Server: `npm run serve-ui`
* Flask Dev Server: `npm run serve-api`

Generate API client/server: `npm run new:api -- <apiName>`
